Notable Projects

USC NetCat

I dreaded having to look through 5-7 different websites just to find all the juicy techtalks, career fairs, and networking events occuring at USC. After talking to a few friends, I realized that many people had the same concerns. Enter USC NetCat (short for USC Network Cataloge), a website that centralizes all career/ networking event data at USC into one convenient location. We automatically extract all the career data using Selenium and Beautiful soup, divide them by major and type, and display it on the website.

Norman Lear Center TV Database

This is the website I was hired to create by the Norman Lear Center (NLC). Its a searchable web database of US television transcripts that returns all transcripts along side relevant meta data containing a key word typed in the search bar. I learned a ton about modern webstacks, backend database management, and reading third-party API documentation while building this. It was an amazing learning experience! Additionally, it helped the NLC save a lot of money since they no longer need to pay dozens of part time workers to search transcripts and extract meta data manually. Unfortunately, I can't post the link to the website because the NLC paid for the third-party api's services and they asked not to make the website totally public.
